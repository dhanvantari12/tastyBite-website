# 🍕 TastyBite – Responsive Pizza Website

TastyBite is a responsive pizza restaurant website built using HTML5 and CSS3.  
This project was created as my first front-end web development project to practice layout structuring, modern UI styling, and responsive design principles.

---

## 🚀 Live Demo

🔗 [Click Here to View TastyBite Website] : https://dhanvantari12.github.io/tastyBite-website/

---

## ✨ Features

- Sticky navigation bar
- Smooth scrolling between sections
- Responsive design (mobile-friendly layout)
- Modern warm color palette
- Interactive hover effects
- Clean and structured UI sections (Home, About, Menu, Contact)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Flexbox
- Media Queries
- Google Fonts
